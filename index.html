Add to Home Screen or Install

To reduce your double work, follow these steps:

For Mobile Devices (Google Chrome)

  1. Tap the menu icon (three vertical dots) in the top right corner.
  2. Select "Add to Home screen" from the menu.
  3. Tap "Add" to confirm.

For Desktop (Google Chrome)

  1. Click the three-dot menu icon in the top right corner.
  2. Select "Create shortcut / Install page as app" from the menu.
  3. Click "Create / Install" to confirm.
